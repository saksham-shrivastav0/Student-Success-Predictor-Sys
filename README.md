# 🎓 Student Performance Prediction System

## 🚀 Overview

This is my **first end-to-end Machine Learning project**, where I built a system to predict a student's performance based on various factors like study hours, attendance, assignments, and previous scores.

The project covers the complete ML pipeline — from data preprocessing to model building and prediction.

---

## 📊 Problem Statement

* Predict **Final Score** of a student (Regression)
* Predict **Performance Level** (High / Medium / Low) (Classification)

---

## 🧠 Features Used

* Previous Score
* Study Hours
* Attendance (%)
* Assignments Completed
* (Optional) City (One-Hot Encoded)

---

## ⚙️ Tech Stack

* Python 🐍
* NumPy
* Pandas
* Matplotlib
* Scikit-learn

---

## 🔄 ML Pipeline

1. Data Loading
2. Data Understanding (`info()`, `describe()`)
3. Data Cleaning
4. Feature Engineering
5. Encoding (Label Encoding + One-Hot Encoding)
6. Train-Test Split
7. Feature Scaling (StandardScaler)
8. Model Training

   * Linear Regression (for score prediction)
   * KNN Classifier (for performance level)
9. Prediction using user input
10. (Optional) Model Evaluation

---

## 🔍 Key Learnings

* Importance of **Train/Test Split**
* Difference between **fit() and transform()**
* Avoiding **Data Leakage**
* Feature selection matters
* Building a **complete ML pipeline**

---

## 🎯 How It Works

1. User inputs:

   * Study Hours
   * Attendance
   * Assignments Completed
   * Previous Score

2. Model predicts:

   * Final Score
   * Performance Level
